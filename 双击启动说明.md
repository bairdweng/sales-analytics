# 🚀 双击启动项目说明

## 方法1：使用终端命令（推荐）

### 启动项目
```bash
# 在Finder中双击 start.sh 文件
# 或者在终端中执行：
./start.sh
```

### 停止项目
```bash
# 在Finder中双击 stop.sh 文件
# 或者在终端中执行：
./stop.sh
```

## 方法2：创建Automator应用程序（双击运行）

### 创建启动应用程序：
1. 打开"自动操作"（Automator）应用
2. 选择"应用程序"
3. 在左侧搜索"运行Shell脚本"
4. 拖拽到右侧工作区
5. 输入以下脚本：
```bash
cd /Users/bairdweng/Desktop/ai-tools/ebay_xp
./start.sh
```
6. 保存为"启动项目.app"

### 创建停止应用程序：
1. 同样步骤，输入以下脚本：
```bash
cd /Users/bairdweng/Desktop/ai-tools/ebay_xp
./stop.sh
```
2. 保存为"停止项目.app"

## 方法3：使用现有的脚本文件

### 让.sh文件可双击执行：
1. 右键点击 `start.sh` 文件
2. 选择"显示简介"
3. 在"打开方式"中选择"终端"
4. 勾选"所有类似文件"
5. 同样设置 `stop.sh` 文件

## 📋 当前可用的启动方式：

1. **终端命令**：`npm start` 或 `./start.sh`
2. **双击脚本**：双击 `start.sh` 和 `stop.sh`（需要设置打开方式）
3. **AppleScript**：双击 `启动项目.applescript` 和 `停止项目.applescript`

## 💡 提示：

- 脚本已经设置为可执行权限
- 启动时会自动检查并安装依赖
- 停止脚本会安全地关闭所有相关进程
- 建议将创建好的应用程序拖到Dock栏方便使用
